# SW동행 데모데이 팀프로젝트

## 프로젝트 소개

**경포호 수질 모니터링 시스템**은 실시간 수질 데이터를 시각화하고 지역별 현황을 제공하는 웹 애플리케이션입니다.

### 주요 기능

- **실시간 수질 데이터 시각화**
- **지역별 수질 현황 모니터링**  
- **시계열 데이터 분석 및 차트 제공**
- **인터랙티브 지도 기반 정보 제공**
