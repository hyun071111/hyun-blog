# Express + MongoDB 게시판

## 프로젝트 소개

Express.js와 MongoDB를 활용하여 개발한 개인 게시판 웹 애플리케이션입니다.

## 사용 기술

| **프론트엔드** | HTML/CSS |
| **백엔드** | Express.js |
| **데이터베이스** | MongoDB |
| **배포** | AWS EC2 |

## 주요 기능

### 게시글 CRUD
- **목록 조회**: MongoDB에서 모든 게시글을 불러와 렌더링
- **글 작성**: 제목/내용 입력 후 POST 요청으로 새 문서 삽입
- **글 수정**: 기존 글 로드 후 수정 내용 업데이트
- **글 삭제**: 해당 ID로 MongoDB 문서 삭제
