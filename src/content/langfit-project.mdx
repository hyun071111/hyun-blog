# 웹/앱 경진대회 LANGFIT

## 프로젝트 개요

**LANGFIT**은 교내 웹/앱 경진대회에서 개발한 MBTI 기반 프로그래밍 언어 추천 서비스입니다. AI 챗봇을 통한 학습 지원과 개인 맞춤형 강의 추천 기능을 제공합니다.

### 주요 기능

- **MBTI 기반 프로그래밍 언어 추천**
- **AI 챗봇을 통한 실시간 학습 지원**
- **인터랙티브 학습 평가 시스템**
- **개인 맞춤형 학습 경로 제공**
